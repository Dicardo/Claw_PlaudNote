# PlaudNote 功能清单

## 核心录音功能
1. **录音质量设置** - 高/中/低质量选择
2. **录音格式选择** - WAV, M4A, MP3
3. **后台录音** - 锁屏/后台继续录音
4. **录音暂停/继续** - 支持分段录音
5. **录音计时器** - 定时自动停止录音
6. **录音增益调节** - 手动调节麦克风灵敏度
7. **降噪功能** - 环境噪音消除

## AI 转录功能
8. **多语言支持** - 中英日韩等语言识别
9. **说话人分离** - 区分不同说话人
10. **实时转录** - 边说边出文字
11. **离线转录** - 本地语音识别（隐私模式）
12. **转录编辑** - 手动修正识别结果
13. **关键词高亮** - 自动标记重要词汇

## AI 摘要功能
14. **会议纪要生成** - 自动提取会议要点
15. **待办事项提取** - 从录音中提取 Action Items
16. **智能分段** - 按话题自动分段
17. **摘要模板** - 会议/采访/课堂/笔记等模板
18. **多格式导出** - TXT, PDF, DOCX, Markdown

## 文件管理
19. **文件夹管理** - 创建/重命名/删除文件夹
20. **标签系统** - 自定义标签分类
21. **搜索功能** - 全文搜索录音内容
22. **收藏功能** - 标记重要录音
23. **批量操作** - 批量删除/移动/导出
24. **排序筛选** - 按时间/时长/名称排序
25. **回收站** - 误删恢复

## 安全与隐私
26. **生物识别** - Face ID / Touch ID 解锁
27. **隐私模式** - 隐藏敏感录音
28. **加密存储** - 本地文件加密
29. **自动锁定** - 离开应用自动锁定

## 数据同步与备份
30. **iCloud 同步** - 多设备同步
31. **自动备份** - 定期备份到云端
32. **导出分享** - 分享到微信/邮件/云盘
33. **导入功能** - 导入外部音频文件

## 统计与分析
34. **使用统计** - 录音时长/数量统计
35. **词频分析** - 高频词汇统计
36. **时间轴视图** - 按日历查看录音

## 快捷操作
37. **Siri 快捷指令** - "开始录音"语音命令
38. **桌面小组件** - 一键录音小组件
39. **键盘快捷键** - iPad 外接键盘支持
40. **3D Touch** - 长按快捷菜单

## 外观与体验
41. **深色模式** - 自动/手动切换
42. **主题颜色** - 多种配色方案
43. **字体大小** - 可调节阅读字体
44. **多语言界面** - 中英文切换

## 协作功能
45. **团队共享** - 录音分享给团队成员
46. **评论批注** - 在转录上添加批注
47. **版本历史** - 转录修改记录

## 高级功能
48. **实时协作** - 多人同时编辑转录
49. **语音指令** - 语音控制录音
50. **智能提醒** - 根据录音内容提醒事项
