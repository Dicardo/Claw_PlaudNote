# PlaudNote Web - AI 录音笔

一个功能完整的 AI 录音笔 Web 应用，支持录音、音频上传、AI 转录、会议纪要生成和任务提取。

## 功能特性

### 1. 录音功能
- 🎙️ 使用 Web Audio API 进行高质量录音
- 📊 实时音频可视化波形显示
- ⏸️ 支持暂停/继续录音
- ⏱️ 录音时长计时显示

### 2. 音频管理
- 📁 本地音频文件上传
- ▶️ 内置音频播放器
- 🗂️ 录音列表管理
- 🗑️ 删除录音功能

### 3. AI 转录
- 📝 自动语音识别转录
- ⏳ 实时转录状态显示
- 📄 转录文本查看
- 💾 转录结果本地存储

### 4. 会议纪要
- 📋 自动生成会议摘要
- 🔑 提取关键要点
- ✅ 识别决议事项
- 📌 整理后续行动

### 5. 任务面板
- 📊 总任务面板（所有任务）
- 👤 个人任务面板
- ✅ 任务完成状态切换
- 🗑️ 任务删除功能

### 6. 数据存储
- 💾 LocalStorage 本地持久化
- 🔄 数据自动保存
- 🗑️ 一键清空数据

## 技术栈

- **前端**: 原生 HTML5 + CSS3 + JavaScript (ES6+)
- **样式**: 现代暗色主题，响应式设计
- **图标**: Font Awesome 6.4.0 CDN
- **API**: Web Audio API, MediaRecorder API

## 如何运行

### 方法 1: 直接打开
1. 使用现代浏览器（Chrome、Firefox、Edge、Safari）
2. 直接打开 `index.html` 文件

### 方法 2: 本地服务器（推荐）
```bash
# 使用 Python 3
cd PlaudNote-Web
python -m http.server 8080

# 或使用 Node.js
npx serve .

# 然后访问 http://localhost:8080
```

### 方法 3: 使用 OpenClaw 网关
```bash
# 启动网关服务
openclaw gateway start

# 应用将自动通过网关访问
```

## 使用说明

### 录音
1. 点击"开始录音"按钮
2. 允许浏览器使用麦克风权限
3. 录音过程中可以看到实时波形
4. 点击"暂停"可暂停录音，"继续"恢复
5. 点击"停止"结束录音并保存

### 上传音频
1. 点击"上传音频文件"按钮
2. 选择本地音频文件（支持 mp3, wav, webm 等格式）
3. 文件上传后会自动开始转录

### 查看转录
1. 切换到"转录文本"页面
2. 点击任意转录项查看详情
3. 可在"转录文本"、"会议纪要"、"提取任务"标签间切换

### 管理任务
1. 切换到"任务面板"页面
2. 在"总任务面板"查看所有任务
3. 在"个人任务面板"查看分配给自己的任务
4. 点击复选框标记任务完成

### 清空数据
点击侧边栏底部的"清空数据"按钮可删除所有录音、转录和任务。
